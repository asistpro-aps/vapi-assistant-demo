Demo sayfanız hazır! Vapi entegrasyonunu tamamlamak için:

  1. Node.js kurulumunu tamamlayın: PATH sorunu nedeniyle React versiyonu çalışmıyor
  2. Vapi hesabı oluşturun: dashboard.vapi.ai
  3. Public Key ve Asistan ID'lerini alın
  4. src/App.js dosyasında konfigürasyon yapın
  5. npm start ile React uygulamasını başlatın
